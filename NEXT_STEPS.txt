# ============================================================================
# 🆘 TROUBLESHOOTING
# ============================================================================

Problem: "Cannot connect to API"
Solution: Make sure Flask is running: python app.py
         Check health: curl http://localhost:5000/health

Problem: "Models not found" / "Model loading failed"
Solution: Export models from notebook first (STEP 1)
         Verify files exist in ./models/
         Restart Flask app

Problem: "Port already in use"
Solution: python app.py --port 5001
         streamlit run dashboard.py --server.port 8502

Problem: "ModuleNotFoundError: No module named X"
Solution: pip install --upgrade -r requirements-backend.txt
         pip install --upgrade -r requirements-frontend.txt

Problem: "ImportError: cannot import name..."
Solution: Make sure you're in the correct directory
         cd "Grid 2 proto/Theme 1"

For detailed troubleshooting: See SETUP_GUIDE.md
